Collapsible FAQ

What is the DOM? +
The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.
What is an event listener? +
An event listener is a procedure or function in a computer program that waits for an event to occur. Examples of events are user clicks, key presses, or mouse movements.
